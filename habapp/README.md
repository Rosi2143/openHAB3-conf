# install additional packages

* install command
  * `/opt/habapp/bin/pip install <package>`
* packages
  * pydot
  * python-statemachine
  * typing
* get list of installed packages
  * `/opt/habapp/bin/habapp -di`
  * or
  * `/opt/habapp/bin/pip list`
